# Employee-Management-System_java

This project is a desktop-based Employee Management System developed using Java Swing for the user interface and MySQL for database management. It is designed to help organizations manage employee records efficiently with a simple and user-friendly interface.

The system allows users to perform basic operations such as adding, updating, viewing, and deleting employee details. It also includes a login system to ensure secure access.

 Features:-
User authentication (Login system)
Add new employee details
View employee records in a tabular format
Update existing employee information
Remove employee records
Simple and interactive GUI using Java Swing
Database connectivity using MySQL

Technologies Used:-
Java (Swing, AWT)
MySQL Database
JDBC (Java Database Connectivity)
📂 Project Structure
Login.java – Handles user authentication
Home.java – Main dashboard of the system
AddEmployee.java – Form to add new employees
ViewEmployee.java – Displays employee data
UpdateEmployee.java – Updates employee details
RemoveEmployee.java – Deletes employee records
Conn.java – Database connection setup
Splash.java – Initial loading screen
